About & Manual

Facilities Automation is an IOT solution for a home security system. This iot solution has an emulation system where we can see people entering and exiting from a home with the help of security door sensor. We also get notified if someone enter with a wrong door pin. Smoke detector sensor used to detect smoke or fire. fire water sprinkle, danger alarm and light actuator are used to fulfill our facility.

  • Click Start smoking button to produce smoke or fire
  • Enter a wrong pin code in text field and click in confirm identity
  • Enter a correct pin code (13579) to stop danger alarm
